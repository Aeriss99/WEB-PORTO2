    OpsFlow Service Desk

    Web aplikasi ticketing internal untuk tim operasional dalam menangani issue lintas divisi. Sistem ini membantu pencatatan, prioritisasi, dan monitoring SLA agar proses support lebih terukur. Dashboard ringkas memudahkan lead memantau beban kerja harian.

    Type: Case Study

    Role: Full-Stack Developer

    Tech Stack: React, Node.js, Express, MySQL, JWT

    Status: In Progress

    Key Features:

    • Role-based access (Agent, Supervisor, Admin).
    • Ticket lifecycle tracking dengan status dan prioritas.
    • SLA timer dan notifikasi untuk ticket overdue.
    • Dashboard performa ticket per tim dan periode.

    Future Improvement: Menambahkan auto-assignment berbasis kategori issue, audit log aktivitas, dan ekspor laporan bulanan ke format CSV/PDF.

    StockPoint Inventory

    Aplikasi inventori untuk bisnis retail skala kecil hingga menengah dengan fokus akurasi stok. Pengguna dapat mencatat barang masuk-keluar, memantau pergerakan stok, dan melihat ringkasan nilai inventori. Proyek ini dirancang sebagai simulasi kebutuhan client UMKM.

    Type: Personal

    Role: Front-End Developer

    Tech Stack: React, Tailwind CSS, Axios, Chart.js

    Status: Completed

    Key Features:

    • Manajemen katalog produk dengan kategori dan SKU.
    • Pencatatan transaksi stok masuk dan stok keluar.
    • Alert stok minimum untuk item kritis.
    • Visualisasi tren stok menggunakan grafik mingguan.

    Future Improvement: Integrasi barcode scanner, sinkronisasi multi-cabang, dan histori harga beli untuk analisis margin.

    TalentBridge Hiring Portal

    Platform rekrutmen sederhana untuk membantu HR mempublikasikan lowongan dan memproses kandidat. Candidate flow dibuat jelas mulai dari apply, screening, hingga interview shortlist. Proyek ini fokus pada pengalaman pengguna dan struktur data yang mudah dikembangkan.

    Type: Concept

    Role: Back-End Developer

    Tech Stack: Node.js, Express, MySQL, REST API, Postman

    Status: Prototype

    Key Features:

    • Manajemen lowongan kerja dan detail kualifikasi.
    • Endpoint lamaran kandidat dengan validasi data dasar.
    • Tahapan seleksi kandidat (applied, screening, interview).
    • Filter kandidat berdasarkan status dan posisi.

    Future Improvement: Menambahkan authentication untuk recruiter, upload CV terstruktur, serta integrasi email notifikasi status lamaran.
